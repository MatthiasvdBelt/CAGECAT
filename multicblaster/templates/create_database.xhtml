This workflow will allow you to create a local database for cblaster from a collection of genome files in GenBank or GFF3+FASTA formats. It will produce a formatted DIAMOND database (.dmnd) containing protein sequences in the supplied genomes, as well as a sqlite3 file which stores genomic coordinates for these proteins. To use these files in a cblaster run, choose a local search, then supply the DIAMOND (.dmnd) file using the Database field.
